# Voyager Excel Export

a plugin for excel export 

## Install hook

```bash
php artisan hook:install tu6ge/voyager-excel
```

## Enable hook

```bash
php artisan hook:enable tu6ge/voyager-excel
```

## Support Language

- zh_CN （中文）
- en
