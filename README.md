# Voyager Excel Export

a plugin for excel export 

## Install

```bash
composer require tu6ge/voyager-excel
```

## Enable

```bash
php artisan vendor:publish --provider="Tu6ge\VoyagerExcel\VoyagerExcelServiceProvider"
```

## Support Language

- zh_CN （中文）
- en
